# 受聘僱外國人入國後健康檢查作業規範學習指南

本指南旨在協助使用者深入理解「受聘僱外國人入國後健康檢查作業規範」的核心內容，透過重點整理、練習題及詞彙解釋，全面掌握相關規定。

---

## 第一部分：核心概念摘要

本規範旨在確保受聘僱外國人入國後健康檢查的品質。其核心內容涵蓋人員資格、檢查流程、各項檢查的具體規範、紀錄保存及通報機制。

### 一、 核心原則與流程

1.  **品質確保**：規範的根本目的是確保健康檢查的品質。
2.  **人員資格**：對執行各項檢查的醫事人員有明確的專業資格要求，例如胸部X光判讀需由放射線專科或胸腔專科醫師執行，漢生病檢查需由特定資格醫師執行，檢驗人員需具備醫檢師證照。
3.  **流程自主性**：健檢醫院需自行完成從報到、檢查到報告核發的全部流程，不得委託第三方（如醫療顧問公司）辦理。
4.  **身分識別**：建立嚴謹的受檢者身分識別機制，於檢查及採檢前核對姓名、相片與護照號碼，並在健檢表單上張貼蓋有騎縫章的相片。
5.  **檢體管理**：建立標準化的檢體採集、標示、運送、保存及監管鏈程序，防止檢體攙假或調換。
6.  **報告時效**：健檢報告應於健檢日次日起十日內核發。若結果不合格或需複檢，雇主需依規定時程安排後續事宜。

### 二、 主要檢查項目規範摘要

| 檢查項目 | 檢查方法 | 合格標準 | 不合格處理與複檢 |
| :--- | :--- | :--- | :--- |
| **胸部X光肺結核檢查** | 大片或數位影像 (14x14in / 14x17in)。妊娠婦女得以三套痰塗片檢查替代。 | 正常、纖維化（鈣化）病灶或肋膜增厚，且系列追蹤無變化、痰液未檢出結核菌。 | 肺結核或結核性肋膜炎視為不合格。疑似個案需於15日內至指定機構再檢查。非多重抗藥性個案可申請都治服務，完成治療者視為合格。 |
| **漢生病檢查** | 醫師在充足光源下目視檢查皮膚。 | 皮膚無疑似漢生病病灶。 | 疑似病灶需於15日內至指定機構再檢查。若同時具備「持續性皮膚病灶感覺喪失/改變或神經腫大」及「皮膚抹片發現麻風桿菌或組織病理符合」兩條件，則為不合格。可申請都治服務。 |
| **腸內寄生蟲糞便檢查** | 採用加裝目鏡測微器的顯微鏡檢查法，檢體需經離心濃縮法前處理。 | 未檢出不合格之原蟲或蠕蟲蟲卵。人芽囊原蟲及部分阿米巴原蟲視為合格。 | 檢出腸道蠕蟲蟲卵或其他致病性原蟲為不合格。非痢疾阿米巴原蟲者，可於65日內治療後再檢。疑似痢疾阿米巴原蟲者，需送疾病管制署確認。 |
| **梅毒血清檢驗** | 結合非特異性試驗 (RPR/VDRL) 與特異性試驗 (TPHA/TPPA等)。 | 兩項試驗結果均為陰性，或治療後效價未達≧4倍上升。 | 未治療者兩項試驗均陽性，或已治療者效價≧4倍上升，視為不合格。可於30日內取得完成治療證明。 |
| **麻疹、德國麻疹疫苗接種/抗體檢驗** | 針對短期補習班外師及移工。接種MMR或MR疫苗，或檢測IgG抗體。 | 檢具疫苗接種證明、抗體陽性證明，或經醫師評估有接種禁忌者。 | 抗體檢驗結果為陰性或未確定者為不合格，應接受疫苗注射。 |

### 三、 紀錄保存與通報

*   **紀錄詳實**：所有檢查、問診、檢驗紀錄皆需詳實，並可追溯至負責人員。
*   **保存年限**：
    *   健康檢查證明、病歷、X光片：至少保存 **7年**。
    *   檢驗結果紀錄、檢驗報告、品管紀錄：至少保存 **3年**。
*   **通報機制**：
    *   指定醫院需依規定時程將健檢結果清單上傳至衛生主管機關指定資訊平臺。
    *   **入國3日內健檢**：受理後次日起 **7個工作日** 內上傳。
    *   **定期或補充健檢**：受理後次日起 **10個工作日** 內上傳。
    *   若檢查結果符合傳染病通報定義，應依《傳染病防治法》完成通報。

3.  為防止檢體被攙假或調換，規範中訂定了哪些具體的程序與機制？
4.  糞便檢體與血清檢體若無法於採集當日檢驗，應如何保存？
5.  胸部X光檢查若診斷為「疑似肺結核」，後續的標準處理流程為何？
6.  妊娠婦女若無法進行胸部X光檢查，應採取何種替代方案？其不合格的判定標準為何？
7.  經確診為肺結核的受聘僱外國人，在何種條件下可以申請「都治服務」？申請流程為何？
8.  漢生病檢查的不合格標準為何？（需列出兩項必備條件）
9.  在腸內寄生蟲糞便檢查中，若檢驗結果為「疑似痢疾阿米巴原蟲」，醫院應採取的後續步驟有哪些？
10. 梅毒血清檢驗中，如何判定一位「曾經接受梅毒治療者」的檢查結果為不合格？
11. 哪些對象需要實施麻疹、德國麻疹疫苗接種或抗體檢驗？在什麼情況下可以豁免？
12. 根據規範，各類醫療紀錄（如病歷、X光片、檢驗報告）的法定最短保存年限分別是多久？
13. 指定醫院應在受理移工「入國後三日內健康檢查」後的幾天內，將結果上傳至指定資訊平臺？

## 第三部分：申論題

1.  請詳細闡述「受聘僱外國人入國後健康檢查作業規範」如何從**人員資格、流程管理、檢體監控**三個面向，建立一套確保健檢品質的綜合管理體系。
2.  請比較肺結核、漢生病與痢疾阿米巴這三項疾病在健康檢查中的**不合格判定標準、複檢要求、以及治療後轉為合格的途徑與條件**。三者在管理策略上有何異同之處？
3.  試論述本規範中「紀錄保存」與「通報衛生機關」兩大章節的重要性。這兩項要求如何支持傳染病防治的公共衛生目標，並確保雇主與受聘僱外國人的權益？

## 第四部分：重要詞彙解釋

| 詞彙 | 解釋 |
| :--- | :--- |
| **都治服務 (DOTS)** | 直接觀察治療 (Directly Observed Treatment, Short-course) 的簡稱。在本規範中，指肺結核或漢生病不合格個案，可向衛生主管機關申請納入此服務，在衛生單位人員的直接觀察下完成藥物治療。完成治療者可被視為合格。 |
| **騎縫章** | 指蓋在文件接縫處或相片與文件紙張交接處的印章。在本規範中，用於健檢表單的個人相片上，以防止相片被抽換，確保受檢者身分的真實性。 |
| **檢體監管紀錄鏈** | (Chain of Custody) 一套程序化的文件紀錄，用於追蹤檢體從採集、運送、接收到檢驗的全過程。紀錄內容包含各環節負責人員的簽名，以確保檢體的完整性與可追溯性。 |
| **離心濃縮法** | 腸內寄生蟲糞便檢查的前處理技術。透過離心將糞便檢體中的寄生蟲卵或原蟲囊體集中，以提高在顯微鏡下檢出的機率。規範中建議採「甲醛-乙酸乙酯離心濃縮法」等。 |
| **非特異性梅毒螺旋體試驗** | 用於梅毒初步篩檢的血清學檢驗方法，如快速血漿反應素試驗(RPR)或性病研究實驗室試驗(VDRL)。其原理是檢測人體因梅毒螺旋體感染而產生的反應素抗體。 |
| **特異性梅毒螺旋體試驗** | 用於確認梅毒感染的血清學檢驗方法，如梅毒螺旋體血液凝集試驗(TPHA)或粒子凝集試驗(TPPA)。其原理是直接檢測針對梅毒螺旋體本身的抗體，專一性較高。 |
| **多重抗藥性結核病 (MDR-TB)** | 指由對至少 Isoniazid 和 Rifampin 這兩種最有效的一線抗結核藥物產生抗藥性的結核菌所引起的結核病。根據規範，此類個案不適用於申請都治服務後轉為合格的規定。 |
| **不合格細項代碼** | 用於在健康檢查結果清單中，標準化標示不合格項目的代碼。例如：X01代表胸部X光檢查異常、S01代表梅毒血清檢查不合格、H01代表漢生病檢查不合格。 |
